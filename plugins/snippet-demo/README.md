# Snippet Demo

Este plugin de teste adiciona um snippet pronto na biblioteca do NovaPad.

O que ele faz:
- Cria um snippet chamado `Demo do Plugin`
- Aparece em `Bibliotecas e snippets`
- Serve para validar o carregamento de plugins locais

Como usar:
1. Abra o NovaPad
2. Vá em `Bibliotecas e snippets`
3. Abra `Seus snippets`
4. Clique em `Demo do Plugin`

Observação:
- O plugin adiciona o snippet apenas uma vez
- Ele não altera conteúdo de nota existente
- Se você remover o snippet manualmente, ele pode voltar na próxima inicialização enquanto o plugin estiver instalado
